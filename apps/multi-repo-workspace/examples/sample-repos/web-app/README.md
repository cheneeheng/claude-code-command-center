# web-app (sample repo)

A stand-in target repo for the roundtable runnable example — it is **not** a real application. Its only job is to carry a couple of plans under `.agents_workspace/planning/` so roundtable has something to list on the board.

It is deliberately not a git repository, so its board card shows a git warning strip — that exercises roundtable's degraded-repo rendering. In real use this would be one of your own git repos. See [`../../README.md`](../../README.md) for how to launch the example.
