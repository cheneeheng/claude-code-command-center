# api-service (sample repo)

A stand-in target repo for the docket runnable example — it is **not** a real
service. Its only job is to carry a plan under `.agents_workspace/planning/` so
docket has a second project to list (and to group a batch by).

In real use this would be one of your own git repos. See
[`../../README.md`](../../README.md) for how to launch the example.
